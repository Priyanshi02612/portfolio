# Priyanshi Kalsariya | Full Stack Web Developer Portfolio

A modern, high-performance developer portfolio designed with a dark, polished aesthetic. This project showcases professional experience, technical skills, and key projects through an interactive interface.

## Key Features

- **Immersive UX**: Smooth page transitions and layout animations powered by Motion.
- **Responsive Design**: Fully optimized for all devices using Tailwind CSS 4.
- **Interactive Project Showcase**: Detailed project cards with technology tags and demo links.
- **Dynamic Content**: Centralized data management via `constants.ts` for easy updates.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4, Lucide Icons
- **Animation**: Motion

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Priyanshi02612/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## Project Structure

- `src/constants.ts`: Central source of truth for all portfolio data.
- `src/App.tsx`: Main application layout and sections.
- `src/index.css`: Global styles and Tailwind configuration.
